Welcome to snekBOL's documentation!
===================================

snekBOL is a python library for the reading and writing of Synthetic Biology Open Language (SBOL)
2.1.0 files. This is a Python 3 implementation of the standard and does not use a wrapper like
pySBOL.

For more details on the SBOL standard see `the official site <http://sbolstandard.org/>`_.
